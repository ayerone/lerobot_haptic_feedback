# Force Feedback for LeRobot SO-101 teleoperator
This project extends the SO-101 Leader (teleoperator) / Follower (robot) by adding haptic feedback.

A force sensor is added to the gripper of the robot, and a brushless "gimbal" motor is used as the final joint of the leader to provide force feedback to the operator during teleoperation.  This allows the operator to use appropriate gripping force, protecting delicate objects and avoiding stress/heat in the robot's gripper motor.
